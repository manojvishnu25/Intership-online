# 🔐 Password Strength Analyzer

A modern and interactive Password Strength Analyzer that evaluates password security in real-time, provides improvement suggestions, generates strong passwords, and helps users learn password security best practices.

---

## 📖 Overview

The Password Strength Analyzer is designed to help users create secure passwords by analyzing password complexity and providing instant feedback. It evaluates password strength based on length, uppercase letters, lowercase letters, numbers, and special characters.

The application also includes a secure password generator, password history tracking, and educational security recommendations.

---

## ✨ Features

### 🔍 Real-Time Password Analysis

* Instant password strength evaluation
* Dynamic strength meter visualization
* Strength categories:

  * Very Weak
  * Weak
  * Fair
  * Good
  * Strong
  * Very Strong

### ✅ Password Requirement Validation

Checks for:

* Minimum 8 characters
* Uppercase letters (A-Z)
* Lowercase letters (a-z)
* Numbers (0-9)
* Special characters (!@#$%^&*)

### 💡 Smart Suggestions

Provides recommendations such as:

* Add uppercase letters
* Include numbers
* Use special characters
* Increase password length
* Avoid repeated characters

### 🔐 Strong Password Generator

* Generates secure random passwords
* Includes all character categories
* One-click password generation

### 📋 Copy to Clipboard

* Easily copy generated passwords
* Toast notification after successful copy

### 📚 Password History

* Stores recent password analyses
* Displays strength level history
* LocalStorage-based persistence

### 👁️ Password Visibility Toggle

* Show/Hide password functionality
* Enhanced user experience

### 🛡️ Security Tips Section

Provides password security best practices including:

* Unique passwords
* Password managers
* Two-Factor Authentication
* Avoiding personal information
